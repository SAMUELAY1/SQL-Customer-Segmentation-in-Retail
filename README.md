#  SQL Project: Customer Segmentation (RFM Analysis)

This project performs RFM (Recency, Frequency, Monetary) analysis using SQL on an online retail dataset.

### Objectives
- Segment customers based on purchase behavior.
- Identify top buyers, inactive users, and loyal customers.

### Technologies
- Python file for data cleaning 
- MySQL Workbench
- SQL Window Functions
- Common Table Expressions (CTEs)

### Files
- `rfm_customer_segmentation.sql` — SQL script for the RFM analysis.
- `retail_data_sample.csv` — Source dataset.

### Key Insights
- Champions: High spenders and frequent recent buyers.
- Loyal: Regular customers with consistent purchases.
- Lost: Customers who haven’t purchased in a long time.
